# 飞书多维表格工单系统配置指南

> 零代码搭建工单管理系统，30 分钟完成配置
> 对应文档：产品化方案/工单全生命周期管理方案.md 第三节

---

## 一、飞书多维表格创建

### 1.1 创建表格
1. 打开飞书 → 新建 → 多维表格
2. 命名：`AI客服工单管理`

### 1.2 字段配置

| 字段名 | 类型 | 配置说明 |
|--------|------|---------|
| 工单编号 | 文本 | 主键，自动生成（TK-20260710-0001） |
| 创建时间 | 日期 | 格式：YYYY-MM-DD HH:MM |
| 工单类型 | 单选 | 选项：退换货、投诉、物流异常、其他 |
| 优先级 | 单选 | 选项：紧急、高、中、低 |
| 问题摘要 | 文本 | AI 自动生成 |
| 用户标识 | 文本 | 脱敏后（如 u_***1234） |
| 处理状态 | 单选 | 选项：待分配、处理中、已解决、已关闭、已重新打开 |
| 处理人 | 人员 | 飞书人员字段 |
| 分配时间 | 日期 | 处理状态变为"处理中"的时间 |
| 解决时间 | 日期 | 处理状态变为"已解决"的时间 |
| SLA截止时间 | 日期 | 根据优先级自动计算 |
| 建议处理方式 | 文本 | AI 自动生成 |
| 处理备注 | 文本 | 人工填写 |
| 关联订单号 | 文本 | 如有 |
| 满意度 | 数字 | 1-5 分 |
| 对话摘要 | 文本 | 脱敏后的对话快照 |

### 1.3 创建视图

| 视图名 | 类型 | 筛选条件 | 用途 |
|--------|------|---------|------|
| 全部工单 | 表格 | 无 | 所有工单总览 |
| 待处理 | 看板 | 处理状态=待分配 | 按优先级看板展示 |
| 处理中 | 表格 | 处理状态=处理中 | 客服工作台 |
| 今日新建 | 表格 | 创建时间=今天 | 每日新建工单 |
| 已超时 | 表格 | SLA截止时间<现在 AND 处理状态!=已关闭 | 超时预警 |

## 二、飞书 API 对接

### 2.1 获取凭证

1. 打开 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 获取 `app_id` 和 `app_secret`
4. 开启权限：
   - `bitable:app`（多维表格读写）
   - `im:message`（发送消息通知，可选）
5. 发布应用并获取审批

### 2.2 获取表格 ID

1. 打开多维表格，URL 中的参数：
   - `baseId` 或 URL 路径中的第一个 ID → `app_token`
   - 切换到目标数据表，URL 中的 `table` 参数 → `table_id`
2. 示例 URL：`https://xxx.feishu.cn/base/ABC123?table=tblXXXX`
   - `app_token` = `ABC123`
   - `table_id` = `tblXXXX`

### 2.3 配置到 Dify

在 Dify 工作流中添加 **HTTP 请求节点**，放在工单创建 Code 节点之后：

| 配置项 | 值 |
|--------|-----|
| 方法 | POST |
| URL | `https://open.feishu.cn/open-apis/bitable/v1/apps/{app_token}/tables/{table_id}/records` |
| 请求头 | `Authorization: Bearer {tenant_access_token}` |
| | `Content-Type: application/json` |
| 请求体 | `{{工单创建节点.feishu_fields_json}}` |

## 三、通知配置（可选）

### 3.1 新工单通知

当新工单创建后，通过飞书机器人发送通知到指定群聊：

```
POST https://open.feishu.cn/open-apis/im/v1/messages?receive_id_type=chat_id
{
  "receive_id": "oc_xxxxx",
  "msg_type": "interactive",
  "content": "{\"header\":{\"title\":{\"tag\":\"plain_text\",\"content\":\"新工单提醒\"}},\"elements\":[{\"tag\":\"div\",\"text\":{\"tag\":\"lark_md\",\"content\":\"工单编号：TK-xxx\\n类型：退换货\\n优先级：中\\n摘要：用户申请退货退款\"}}]}"
}
```

### 3.2 SLA 超时提醒

通过飞书多维表格的「自动化」功能：
1. 在表格中点击「自动化」
2. 触发条件：SLA截止时间 < 当前时间 AND 处理状态 ≠ 已关闭
3. 执行动作：发送消息到指定群聊 / @处理人

## 四、数据看板

### 4.1 使用飞书多维表格内置图表

在表格中创建「仪表盘」视图，添加：

| 图表类型 | 数据维度 | 用途 |
|---------|---------|------|
| 数字卡片 | 今日新建工单数 | 流量概览 |
| 饼图 | 工单类型分布 | 问题分类占比 |
| 柱状图 | 处理人工作量 | 负载均衡 |
| 折线图 | 日均工单量趋势 | 趋势监控 |
| 数字卡片 | 平均响应时长 | 效率监控 |

### 4.2 核心指标公式

| 指标 | 飞书公式 |
|------|---------|
| 响应时长 | `DATETIME_DIFF([分配时间], [创建时间], "min")` |
| 处理时长 | `DATETIME_DIFF([解决时间], [创建时间], "min")` |
| SLA 状态 | `IF([SLA截止时间] < NOW(), "已超时", IF(DATETIME_DIFF([SLA截止时间], NOW(), "h") <= 1, "即将超时", "正常"))` |
